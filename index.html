Publications

  1. Output Feedback Periodic Event-Triggered and Self-Triggered Boundary Control of Coupled 2 × 2 Linear Hyperbolic PDEs.
    Eranda Somathilake; Bhathiya Rathnayake; Mamadou Diagne.
    Automatica, 2025. [paper] [arXiv] [bibTeX]

    An observer-based periodic event-triggered and self-triggered boundary control schemes are presented for a class of 2 × 2 linear hyperbolic PDEs. The control input is applied in a zero-order hold fashion between events determined by the triggering conditinos. Under periodic event-triggered control, the triggering condition is monitored periodically and under self-triggered control, the next event time is determined at the current event-time.

  2. Output Feedback Periodic Event-triggered Control of Coupled 2 × 2 Linear Hyperbolic PDEs.
    Eranda Somathilake; Bhathiya Rathnayake; Mamadou Diagne.
    American Control Conference, Denver, USA (ACC), 2025. [paper] [bibTeX]

    Proposes an observer-based periodic event-triggered boundart control stratergy for a class of 2 × 2 linear hyperbolic PDEs where the control is updated at the control input is applied in a zero-order hold fashion between events by periodically monitoring a triggering function. This facilitates the controller to be implemented in a digital platform.
